# E-Commerce Product Explorer

## Project Overview
This is a frontend eCommerce web application built using HTML, CSS, and JavaScript. The application fetches product data from a public API and displays it dynamically in a user-friendly interface.

Users can browse products, search for specific items, filter results by category, and sort them based on price or other criteria.

---

## Features
- Search products by name
- Filter products by category
- Sort products (Price: Low to High / High to Low)
- Dynamic product listing using API data
- Fully responsive design (mobile, tablet, desktop)
- Fast and interactive UI

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API

---

## API Used
Fake Store API  
https://fakestoreapi.com/products  

This API provides product data including title, price, description, category, and images.

---

## How It Works
1. The application makes a request to the API using the Fetch API.
2. The data is received in JSON format.
3. JavaScript processes the data using array methods like `map()`, `filter()`, and `sort()`.
4. The UI updates dynamically based on user interactions.
